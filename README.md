# Ghostruner Manual AP
## Requirement
> [GRUnlocker](https://github.com/Dmgvol/GRUnlocker) : 
- Needed to reset the Collectibles as they dissapear once collected => type 15 when prompted
- Access to all levels (if not, use GRUnlocker and type 2 when prompted) 
## Objective
> Complete the level 17 : "The Monster"
## Randomization
### Progression
- Abilities : Blink, Tempest, Surge, Overlord (Progression because needed for Cybervoids that unlock the corresponding ability)
- Prominent Environmental mobility : Ziplines, Hacking Abilities (Pause Gear + Rotate Signs)
- Augments : Slowmo, High Jump, Shuriken
- Progression Mcguffin :
  - Boss Passes (unlocks boss fights) : TOM, Hel, Mara
  - Architect Pass
  - Cybervoid Passes (mcguffins to access levels containing a cybervoid segment)
  - Hel Memory X : give access to the Xth level of the Project_Hel DLC
### Useful
- Abilities Upgrades
### Traps
- Restart Level
- Reload checkpoint (Instant Restart)
## Locations
- Level Completions
- Collectibles : Artifacts, Audio Logs, Swords
